# Desafios de Manipulação do DOM 

Este repositório contém as minhas soluções para os laboratórios práticos de JavaScript e manipulação do DOM da mentoria DevXperience Bootcamp 2.0. 

O objetivo principal destes exercícios é consolidar os conceitos fundamentais de JavaScript no navegador, mantendo o código simples, legível e directo ao ponto.

## Projectos Incluídos

Todos os desafios foram agrupados em um único ficheiro (`index.html`) para facilitar a visualização e os testes práticos:

* **Lab 3 - Sistema de Contador:** Um contador de "likes" interactivo para aplicar a actualização dinâmica da Interface de Usuário (UI).
* **Lab 4 - Simulação de Login:** Um formulário simples com validação de credenciais estáticas via JavaScript.
* **Lab 5 - Mini Todo List:** Uma lista de tarefas dinâmica onde é possível adicionar novos itens e removê-los do DOM.
* **Mini Projeto Final - Sistema de Inscrição:** Validação de formulário baseada em regras condicionais de idade.

## Tecnologias Utilizadas

* **HTML5:** Estruturação semântica das seções.
* **CSS3:** Estilização básica para separação visual dos laboratórios.
* **JavaScript:** Lógica de programação, captura de eventos (`addEventListener`) e manipulação de elementos (`getElementById`, `createElement`).
